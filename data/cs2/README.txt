## data/classroom_sessions/

Place classroom session images here.

### Folder structure:
  <session_id>/
    row1.jpg   (front row)
    row2.jpg   (middle row)
    row3.jpg   (back row)
    ...

### Session ID format:
  Use a meaningful name: YYYY-MM-DD_SubjectCode
  Example: 2024-11-15_CS401

### Tips:
  - 3-4 images per session, one per bench row
  - Consistent camera angle helps (mount camera at front)
  - Aim for face size >= 40x40 pixels in the image
  - JPG or PNG format supported
